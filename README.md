## Projeto

- O aplicativo tem intuito de mostrar o perfil do usuário do github, assim como seus repositórios e sua linha do tempo. Conta também com uma aba para encontrar trabalho pela plataforma do GitJobs de acordo com localização e especificação
  Por conta de bloqueios de CORS da API do GitJobs foi necessário o uso de um proxy online. Dessa maneira é necessário solicitar acesso no link a seguir
  <a href="https://cors-anywhere.herokuapp.com/corsdemo">Clique aqui para solicitar acesso</a>
  <br/>
  Para baixar o APK do aplicativo <a href="https://drive.google.com/uc?export=view&id=1C33iz0wEATxME5mBeLizDuF8o2O-15I-">clique aqui</a>

## Aplicativo

Ao entrar no aplicativo o usuário deve informar qual o nome de usuário do github.
<img src="https://drive.google.com/uc?export=view&id=1RBhWtCWVPsvr5uL4BY5hgUCiy0Ga3gv9" style="width: 650px; max-width: 100%; height: auto" title="Exemplo" />
<br/>
Caso o nome de usuário não exista, informará um erro
<img src="https://drive.google.com/uc?export=view&id=1He0ZTeThbRMNaet9TS5Fsc4Bw78cJ4QT" style="width: 650px; max-width: 100%; height: auto" title="Exemplo" />
<br/>
Após informar o nome de usuário e ser validado, redirecionará para a tela de "Meu perfil" com algumas informações do usuário e um menu de navegação aparecerá
<br/>
<img src="https://drive.google.com/uc?export=view&id=1IyMFXro6LffuPR5-0NzVHWy2ouD5HvAh" style="width: 650px; max-width: 100%; height: auto" title="Exemplo" />
<br/>
Se o usuário selecionar a opção "Find a Job" no menu, aparecerá uma tela onde o usuário pode procurar trabalhos de acordo com a área desejada (desenvolvedor java, python, etc.), localização e se é uma vaga de tempo integral, ou seja definitiva.
<br/>
<img src="https://drive.google.com/uc?export=view&id=12o-BYMrhdCaA1apgvAz9VrKHxr8p_g_f" style="width: 650px; max-width: 100%; height: auto" title="Exemplo" />
<br/>
Se o usuário selecionar a opção "Repositories" no menu, aparecerá uma lista com todos seus repositórios e as informações do mesmo.
<br/>
<img src="https://drive.google.com/uc?export=view&id=1lOLuSC87eLDi2zeoi6iN-dSxddyyrDpG" style="width: 650px; max-width: 100%; height: auto" title="Exemplo" />
<br/>
Se o usuário selecionar a opção "Timeline" aparecerá uma linha do tempo com as atividades pública do usuário.

<img src="https://drive.google.com/uc?export=view&id=1qcLLZoXcdl9OCugXa0yIJdYd-R6nLBI8" style="width: 650px; max-width: 100%; height: auto" title="Exemplo" />
